Приложение состоит из двух страниц.
Первая страница - телефонные записи (общий список).
Вторая страница - форма добавления или редактирования одной записи.

При запуске приложения выводится первая страница.
На ней можно отфильтровать список телефонных записей используя строку поиска.
Поиск выполняется по всем трем полям: фамилия, имя, телефон.

Нажатие на кнопку "Добавить" внизу страницы или на кнопку "Редактировать" напротив существующей записи ведет на вторую страницу.
На ней можно создать новую телефонную запись, отредактировать поля уже существующей записи, вернуться обратно на первую страницу без сохранения изменений.

Правила валидации полей:
Фамилия - только русские буквы, 1-25 символов
Имя - только русские буквы, 1-15 символов
Телефон - только цифры, 1-20 символов

Для оформления приложения используется библиотека Bootstrap.

База данных представлена файлом PhoneBook.mdf и состоит из одной таблицы Record.

Вся основная логика программы находится в контроллере HomeController.

Для экспорта в Excel используется библиотека EPPlus.

Все сторонние библиотеки устанавливались через NuGet. Для запуска приложения может потребоваться включить функцию восстановления пакетов NuGet.
